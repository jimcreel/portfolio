Disney Reservation Checker

The Disney Reservation Checker is a full-stack site with a javascript front end and python back end. The site pulls publicly available reservation data from Disney websites and displays it for the user filtered by resort, park, date, and annual pass. This site features a custom API running on AWS to route the data to the user.
